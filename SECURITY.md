# Security Policy

## Supported Versions

FlutterHelm is currently maintained as a single mainline branch. Security fixes
are expected to land on the latest published stable contract line.

| Version | Supported |
| --- | --- |
| `0.2.x` | Yes |
| pre-`0.2.0` development snapshots | No |

## Reporting a Vulnerability

Please do not open public issues for security problems.

Preferred reporting path:

1. Use GitHub private vulnerability reporting for this repository if it is enabled.
2. If private reporting is not available yet, contact the repository maintainers through the CAPHTECH organization on GitHub and avoid public disclosure until triage is complete.

When reporting, include:

- affected version or commit
- reproduction steps
- impact assessment
- any workaround or mitigation you have already confirmed

We will acknowledge valid reports as quickly as possible and coordinate a fix before public disclosure.
